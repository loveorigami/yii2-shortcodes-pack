# Youtube

## Config

After installiation in plugins system, you can change default config

```php
    [
        'code' => 'ZM2tVuy8B_Y',
        'w' => 560,
        'h' => 315,
        'playerParameters' => [
            'controls' => 2
        ],
    ];
```

## Usage

```php
    [youtube w=600 h=480 code='ZM2tVuy8B_Y']
```
or
```php
    [yt w=600 h=480 code='ZM2tVuy8B_Y'] link for original video [/yt]
```
